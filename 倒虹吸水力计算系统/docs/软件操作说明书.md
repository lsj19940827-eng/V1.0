## 1. 软件概述

### 1.1 软件简介
倒虹吸水力计算软件是一款专业的水利工程设计辅助工具，用于倒虹吸管道的水力计算与设计。软件严格遵循《灌溉与排水工程设计标准》(GB 50288-2018)附录L的计算规范，提供可视化的管道剖面图、详细的水头损失计算和校验功能。

### 1.2 主要功能
- **管道设计计算**：根据设计流量和拟定流速自动计算管径，支持并联管道（1~10根）
- **水头损失计算**：精确计算沿程损失、局部损失和渐变段损失（三段式）
- **多种结构段支持**：直管、弯管、折管、拦污栅、闸门槽、旁通管、管道渐变段等
- **DXF文件导入**：支持平面管线和纵断面DXF导入
- **三维空间合并计算**：自动合并平面和纵断面数据，计算空间长度和空间转角
- **可视化画布**：实时显示管道纵断面/平面视图，支持缩放和拖拽
- **加大流量工况**：同步计算设计流量和加大流量两种工况
- **详细计算过程**：可选输出完整的计算步骤和中间结果
- **结果导出**：支持Word(.docx)、Excel(.xlsx)、TXT三种格式导出
- **多标签页管理**：通过推求水面线模块自动提取多座倒虹吸，批量计算

### 1.3 技术规范依据
- GB 50288-2018《灌溉与排水工程设计标准》附录L
- 采用谢才公式计算沿程损失
- 采用分段求和法计算局部损失
- 三段式水头损失计算：进口渐变段(ΔZ1) + 管身段(ΔZ2) - 出口渐变段(ΔZ3)

## 6. 功能详细说明

### 6.1 全局参数设置

#### 6.1.1 设计流量 Q
- **定义**: 倒虹吸设计流量,单位: m³/s
- **输入范围**: 大于0的正数
- **典型值**: 5~50 m³/s
- **作用**: 用于计算管径和流速

#### 6.1.2 拟定流速 v
- **定义**: 初步拟定的管内流速,单位: m/s
- **输入范围**: 大于0,一般1.5~3.5 m/s
- **推荐值**: 
  - 小型倒虹吸: 1.5~2.5 m/s
  - 大型倒虹吸: 2.0~3.5 m/s
- **说明**: 
  - 流速过小: 管径过大,工程造价高,易淤积
  - 流速过大: 沿程损失大,能耗高,磨损加剧
- **注意**: 当填写"自定义设计管径"时,此项自动禁用

#### 6.1.3 上游水位 & 下游水位
- **定义**: 倒虹吸进出口渠道的设计水位,单位: m
- **输入要求**: 上游水位必须高于下游水位
- **作用**: 计算可用水位差,用于校验水头损失是否满足要求

#### 6.1.4 上游渠底高程
- **定义**: 上游渠道底部高程,单位: m
- **作用**: 
  - 用于绘制管道剖面图
  - 从DXF导入时会自动填充建议值

#### 6.1.5 糙率 n
- **定义**: 管道内壁糙率系数,无量纲
- **默认值**: 0.014
- **常用值**:
  - 混凝土管: 0.013~0.014
  - 钢管: 0.012~0.013
  - 浆砌石: 0.020~0.025
- **作用**: 用于计算谢才系数和沿程损失

#### 6.1.6 自定义设计管径 D
- **定义**: 用户指定的管径,单位: m
- **输入要求**: 大于0的正数,或留空自动计算
- **作用**: 
  - 留空: 软件根据Q和v自动计算管径
  - 填写: 使用指定管径进行校核计算
- **说明**: 
  - 填写后"拟定流速"自动禁用
  - 用于已确定管径的校核计算

### 6.2 渐变段配置

#### 6.2.1 进口渐变段型式
- **可选类型**:
  - 无
  - 反弯扭曲面 (ξ₁=0.10)
  - 1/4圆弧 (ξ₁=0.15)
  - 方头型 (ξ₁=0.30)
  - 直线扭曲面 (ξ₁=0.20)

- **说明**: 选择型式后自动填充对应的局部阻力系数

#### 6.2.2 进口渐变段始端流速 v₁
- **定义**: 进口渐变段起始断面(渠道断面)的平均流速,单位: m/s
- **推荐**: 采用上游渠道断面平均流速
- **计算方法**: v₁ = Q / A渠道
- **作用**: 用于计算进口段水面落差 ΔZ1

#### 6.2.3 进口渐变段末端流速 v₂
- **定义**: 进口渐变段末端断面(管口断面)的流速,单位: m/s
- **v₂策略** (通过下拉框选择):
  - **自动(=管道流速)** (推荐): v₂ = v（管道实际流速），计算后自动回填
  - **v₁ + 0.2**: 经验增量法，v₁变化时自动联动
  - **由断面参数计算**: 双击提示标签打开断面参数对话框，输入B/h/m
  - **手动输入**: 用户直接填写v₂值
- **安全兜底**: 若v₂ ≤ v₁且v₁ > 0，自动回退到管道流速
- **校验**: v₂必须大于v₁（非阻断警告）

#### 6.2.4 出口渐变段配置
- 与进口段类似
- **出口渐变段始端流速 v**: 一般采用管道出口处流速(即管内流速)
- **出口渐变段末端流速 v₃**: 一般采用下游渠道断面平均流速

### 6.3 结构段管理

#### 6.3.1 添加结构段
1. 点击[添加段]按钮
2. 在弹出对话框中设置:
   - **类型**: 选择结构段类型
   - **长度**: 输入管段长度(直管、折管)
   - **拐弯半径R**: 输入弯管半径(仅弯管)
   - **拐角θ**: 输入拐角度数(弯管、折管)
   - **局部系数**: 自动计算或手动输入
3. 点击[确定]完成添加

#### 6.3.2 结构段类型说明

**直管**:
- 参数: 长度L
- 局部系数: 一般为0(无局部损失)
- 用途: 连接段

**弯管**:
- 参数: 弧长L, 半径R, 圆心角θ
- 局部系数: 根据R/D和θ自动查表计算
  - 依据表L.1.4-3(直角弯道损失系数)
  - 依据表L.1.4-4(任意角修正系数)
  - 公式: ξ = ξ₉₀ × γ

**折管**:
- 参数: 总长L, 折角θ
- 局部系数: 根据折角自动计算
  - 公式: ζ = 0.9457×sin²(θ/2) + 2.047×sin⁴(θ/2)

**拦污栅**:
- 需点击[详细配置]按钮设置:
  - 栅面倾角α (度)
  - 计算模式: 无独立支墩/有独立支墩
  - 栅条参数: 形状、厚度s1、间距b1
  - 支墩参数: 形状、厚度s2、净距b2(有支墩时)
- 局部系数计算:
  - 无支墩: ξ = β1×(s1/b1)^(4/3)×sin(α)
  - 有支墩: ξ = [β1×(s1/b1)^(4/3) + β2×(s2/b2)^(4/3)]×sin(α)

**闸门槽**:
- 参数: 局部系数ξ
- 推荐值: 0.05~0.15 (平板门门槽)
- 默认值: 0.1

**旁通管**:
- 用途: 冲沙管、放空管、进人孔等
- 参数: 局部系数ξ
- 推荐值: 0.10
- 说明: 不计入管道长度,仅计算局部损失

**其他**:
- 用途: 其他附属设施
- 参数: 长度L(可选), 局部系数ξ
- 默认值: ξ=0.1

**进水口/出水口**:
- 位置: 固定在首尾,不可删除
- 进水口: 双击设置形状(完全修圆/稍微修圆/未修圆)
- 出水口: 双击设置流入明渠系数
  - 需输入下游明渠参数(底宽B、水深h、坡比m)
  - 自动计算: ξc = (1 - ωg/ωq)²

#### 6.3.3 编辑结构段
- **双击表格行**可编辑该段参数
- **拖拽表格行**可调整结构段顺序(首尾行除外)
- **锁定状态**: 从DXF导入的段为锁定状态,编辑时会提示确认

#### 6.3.4 删除结构段
1. 选中要删除的表格行
2. 点击[删除段]按钮
3. 确认删除
- **注意**: 不能删除进水口和出水口

#### 6.3.5 清空结构段
- 点击[清空]按钮
- 清空所有结构段并恢复默认示例配置

#### 6.3.6 导入DXF文件
1. 点击[导入 DXF]按钮
2. 选择DXF文件(CAD图纸)
3. 软件自动解析:
   - 识别多段线(LWPOLYLINE/POLYLINE)
   - 判断直管、弯管、折管
   - 提取几何参数(长度、半径、角度、坐标)
   - 自动填充上游渠底高程
4. 解析成功后显示导入结果

**DXF文件要求**:
- 必须包含多段线实体
- 多段线代表管道中心线
- 支持凸度(bulge)表示的弯管

### 6.4 计算与结果

#### 6.4.1 执行计算
1. 确认拟定流速已确认（绿色边框，方案D）
2. 确认管道根数已确认
3. 选择是否勾选"考虑加大流量"和"显示详细过程"
4. 点击[计算]按钮
5. 等待计算完成(通常<1秒)
6. 结果自动显示在Tab 4，InfoBar提示计算结果摘要

#### 6.4.2 结果窗口内容
计算完成后弹出结果窗口,显示:

**基本结果**:
- 理论管径: 根据Q和v计算的理论值
- 设计管径: 取整后的设计管径(或用户指定值)
- 断面积: πD²/4
- 管内流速 v: 实际流速
- 进口渐变段始端流速 v₁
- 出口渐变段末端流速 v₃
- 水力半径: D/4
- 谢才系数: C = (1/n)×R^(1/6)

**水头损失分解**:
- 进口渐变段落差 ΔZ1: (1+ξ₁)×(v₂²-v₁²)/(2g)
- 管身段水头损失 ΔZ2: hf + hj
  - 沿程损失 hf: v²L/(C²R)
  - 局部损失 hj: Σξ×v²/(2g)
- 出口渐变段落差 ΔZ3: (1-ξ₂)×(v²-v₃²)/(2g)
- 总水面落差 ΔZ: ΔZ1 + ΔZ2 - ΔZ3

**校验结果**:
- 管道总长: Σ各段长度
- 可用水位差: H上游 - H下游
- 所需水位差: ΔZ
- 校验结果: 通过/失败
- 详细信息: 
  - 通过: 显示安全裕度
  - 失败: 显示差额

**详细计算过程**(可选):
- 步骤1: 几何设计与流速计算
- 步骤2: 阻力参数初始化

- 步骤4: 校验与结果生成

## 7. 计算公式说明

### 7.1 设计截面计算
```
理论管径: D = √(4ω/π) = √(4Q/(πv))
设计管径: 
  D ≤ 1.0m  → 按0.05m取整
  D ≤ 1.6m  → 按0.1m取整
  D > 1.6m  → 按0.2m取整
实际流速: v = Q / A = 4Q/(πD²)
水力半径: R = D/4 (圆管满流)
谢才系数: C = (1/n) × R^(1/6)
```

### 7.2 水头损失计算(依据附录L)

#### 7.2.1 进口渐变段水面落差 ΔZ1
```
公式 L.1.2-2:
ΔZ1 = (1 + ξ₁) × (v₂² - v₁²) / (2g)

其中:
  ξ₁ - 进口局部损失系数(表L.1.2)
  v₁ - 进口渐变段始端流速(渠道流速)
  v₂ - 进口渐变段末端流速(管口流速)
  g - 重力加速度 9.81 m/s²
```

#### 7.2.2 管身段总水头损失 ΔZ2
```
公式 L.1.4-7:
ΔZ2 = hf + hj

沿程损失:
hf = v² × L / (C² × R)

局部损失:
hj = Σξ × v² / (2g)

其中:
  L - 管道总长度
  Σξ - 管身段局部阻力系数和(含弯管、折管、拦污栅等)
```

#### 7.2.3 出口渐变段水面落差 ΔZ3
```
公式 L.1.3-2:
ΔZ3 = (1 - ξ₂) × (v² - v₃²) / (2g)

其中:
  ξ₂ - 出口局部损失系数
  v - 出口渐变段始端流速(管口流速)
  v₃ - 出口渐变段末端流速(渠道流速)
```

#### 7.2.4 总水面落差
```
公式 L.1.6:
ΔZ = ΔZ1 + ΔZ2 - ΔZ3

注意: 出口段水面落差为负值,故用减号
```

### 7.3 局部阻力系数计算

#### 7.3.1 弯管系数(表L.1.4-3, L.1.4-4)
```
步骤:
1. 计算 R/D0 比值
2. 查表L.1.4-3得到直角弯道系数 ξ₉₀ (线性插值)
3. 查表L.1.4-4得到角度修正系数 γ (线性插值)
4. 计算 ξ = ξ₉₀ × γ
```

#### 7.3.2 折管系数
```
公式:
ζ = 0.9457 × sin²(θ/2) + 2.047 × sin⁴(θ/2)

其中: θ为折角(度)
```

#### 7.3.3 拦污栅系数(表L.1.4-1)

**无独立支墩(公式L.1.4-2)**:
```
ξ = β₁ × (s₁/b₁)^(4/3) × sin(α)

其中:
  β₁ - 栅条形状系数(查表L.1.4-1)
  s₁ - 栅条厚度(mm)
  b₁ - 栅条间距(mm)
  α - 栅面倾角(度)
```

**有独立支墩(公式L.1.4-3)**:
```
ξ = [β₁×(s₁/b₁)^(4/3) + β₂×(s₂/b₂)^(4/3)] × sin(α)

其中:
  β₂ - 支墩形状系数
  s₂ - 支墩厚度(mm)
  b₂ - 支墩净距(mm)
```

**形状系数表**(表L.1.4-1):
| 形状名称 | 系数β |
|---------|-------|
| 矩形 | 2.42 |
| 单侧圆头 | 1.83 |
| 圆形 | 1.79 |
| 双侧圆头 | 1.67 |
| 倒梯形单侧圆头 | 1.04 |
| 梨形/流线型 | 0.92 |
| 两端尖锐型 | 0.76 |

#### 7.3.4 出水口系数(流入明渠)
```
ξc = (1 - ωg/ωq)²

其中:
  ωg - 管道断面积 = Q/v
  ωq - 下游明渠断面积 = (B + m×h)×h
  B - 明渠底宽
  h - 明渠水深
  m - 边坡比
```

### 7.4 校验条件
```
校验公式:
可用水位差 ≥ 所需水位差
即: (H上游 - H下游) ≥ ΔZ

校验通过: 安全裕度 = 可用水位差 - ΔZ
校验失败: 差额 = ΔZ - 可用水位差
```

## 9. 常见问题解答

### 9.1 软件使用类

**Q1: 软件无法启动,提示缺少模块?**

A: 检查是否安装了必需的依赖库:
```bash
pip install PySide6 qfluentwidgets ezdxf
```
确保Python版本≥3.8。

**Q2: 点击计算按钮没有反应?**

A: 检查以下几点:
- 设计流量Q是否大于0
- 拟定流速v是否大于0
- 参数输入框是否有非法字符
- 查看命令行是否有错误提示

**Q3: 管道剖面图不显示?**

A: 可能原因:
- 结构段没有坐标信息(手动输入时正常)
- 点击[适应窗口]按钮重新调整视图
- 检查水位参数是否正确

**Q4: 导入DXF失败?**

A: 检查:
- 是否安装了ezdxf库
- DXF文件是否包含多段线(LWPOLYLINE/POLYLINE)
- 文件是否损坏
- 尝试在CAD中重新保存为R2018格式

**Q5: 拦污栅详细配置窗口显示不完整?**

A: 窗口高度为750px,确保显示器分辨率≥1280×720。可拖拽窗口边缘调整大小。

### 9.2 计算结果类

**Q6: 校验失败,提示水位差不足?**

A: 解决方案:
- **方案一**: 增大上下游水位差
- **方案二**: 减小管径(降低流速,减少损失)
- **方案三**: 减少弯管数量或增大弯管半径
- **方案四**: 优化拦污栅设计,减小栅条厚度或增大间距
- **方案五**: 缩短管道长度

**Q7: 计算出的管径过大或过小?**

A: 检查:
- 拟定流速是否合理(1.5~3.5 m/s)
- 设计流量是否正确
- 如需使用指定管径,填写"自定义设计管径"

**Q8: 弯管系数为何变化?**

A: 弯管系数与以下因素有关:
- R/D比值: 半径越大,系数越小
- 弯管角度: 角度越大,系数越大
- 管径: 管径增大,R/D变小,系数增大

**Q9: 详细计算过程中的系数从何而来?**

A: 所有系数均依据GB 50288-2018附录L:
- 渐变段系数: 表L.1.2
- 弯管系数: 表L.1.4-3, L.1.4-4
- 拦污栅系数: 表L.1.4-1, 公式L.1.4-2/L.1.4-3
- 谢才公式: L.1.4
- 水头损失: L.1.2-2, L.1.3-2, L.1.4-7, L.1.6

**Q10: v₂的四种策略如何选择?**

A: 通过基本参数Tab中的v₂策略下拉框选择:
- **自动(推荐)**: v₂=管道流速，计算后自动回填
- **v₁+0.2**: 经验增量法，v₁变化时自动联动v₂
- **断面参数计算**: 双击提示标签输入B/h/m，计算v₂=Q/[(B+m*h)*h]
- **指定输入**: 直接填写v₂值

若v₂≤v₁且v₁>0，系统自动回退到管道流速

### 9.3 操作技巧类

**Q11: 如何快速调整结构段顺序?**

A: 
- 使用工具栏的[上移][下移]按钮调整顺序
- 首行(进水口)和末行(出水口)不能移动
- 平面段(浅蓝色)为只读，不可编辑/删除

**Q12: 如何批量修改参数?**

A: 
- 当前版本需逐个编辑
- 建议: 先删除现有段,然后重新添加

**Q13: 如何保存设计方案?**

A: 
- 通过推求水面线模块的项目保存功能(.siphon.json)自动持久化
- 支持to_dict/from_dict序列化，UI状态完整还原
- 也可导出为Word/Excel/TXT文件保存计算结果

**Q14: 结构段太多,如何查看?**

A: 
- 表格支持垂直滚动
- 管道剖面图可缩放和平移
- 建议: 分段设计,分段计算

**Q15: 如何对比多个方案?**

A: 
- 方案一: 多次计算,导出不同结果文件对比
- 方案二: 记录关键参数(管径、损失、裕度),制表对比

### 9.4 高级功能类

**Q16: 如何使用自定义管径进行校核?**

A: 
1. 在"自定义设计管径"中输入已知管径(如2.5m)
2. 此时"拟定流速"自动禁用
3. 点击计算,软件将:
   - 使用该管径计算实际流速
   - 计算水头损失
   - 校核是否满足水位差要求

**Q17: 拦污栅设计有什么建议?**

A: 
- **栅条形状**: 优先选择流线型或双侧圆头,系数小
- **栅条间距**: 在满足拦污要求前提下,尽量增大间距
- **栅面倾角**: 倾斜放置(α<90°)可减小系数
- **独立支墩**: 非必要不设置,增加阻力
- **栅条厚度**: 尽量减小,但需考虑强度

**Q18: 如何优化设计减小水头损失?**

A: 
- **减少弯管**: 尽量采用直线布置
- **增大弯管半径**: R/D≥4时效果明显
- **避免急弯**: 弯管角度尽量<90°
- **采用折管代替弯管**: 折角<30°时损失较小
- **优化进出口**: 采用完全修圆的进水口
- **提高管道表面质量**: 选用糙率小的材料
- **合理设置流速**: 流速过大导致损失增加

**Q19: 出水口系数如何选择?**

A: 
- **流入明渠**: 使用公式 ξc=(1-ωg/ωq)²,软件可自动计算
- **流入水库**: 采用渐变段型式系数(表L.1.4-5)
- **直接出口**: 使用表L.1.3中的系数

**Q20: 能否计算非圆形管道?**

A: 当前版本仅支持圆形管道。非圆形管道需:
- 转换为等效圆形: D = 4R (R为水力半径)
- 或手工计算后输入等效参数
