# 倒虹吸水力计算软件需求规格说明书 - 拦污栅配置模块

## 1. 模块概述

本模块旨在为用户提供一个可视化的拦污栅参数配置环境。用户无需翻阅纸质规范，直接在软件中对照“规范示意图”和“系数表”选择栅条形状，程序自动提取对应系数并完成水头损失计算。

## 2. 用户界面 (UI) 需求

### 2.1 交互入口

**位置：主窗口 -> 结构段信息 (Tab 2)。**

**触发方式：**

**当表格中某行的“结构类型”被选为 “拦污栅” 时，该行的“局部阻力系数”单元格变为只读。**

**该行出现一个 [详细配置] 按钮（图标建议为“齿轮”或“编辑笔”），或支持双击该行弹出配置窗口。**

## 5. 异常处理

**输入非法：若用户输入 ，实时在结果框显示 “Error” 或红色文字提示“间距不能为0”。**

**系数值覆盖：虽然提供了自动计算，建议在窗口底部增加一个 “强制手动输入” 的复选框。勾选后，禁用所有几何输入框，允许用户直接在一个文本框中输入  最终值。这是为了应对规范之外的特殊异形栅条。**
