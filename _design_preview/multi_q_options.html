多流量输入方案对比

以下4种方案均保留现有的"管材/管长/局部损失/管径/加大流量"等参数不变，仅改造Q输入区域

A 逗号分隔输入

在现有Q输入框中直接输入多个值，用逗号或空格分隔。最小改动，零学习成本。

设计流量 Q (m³/s):
支持逗号或空格分隔多个流量值，如: 0.21, 0.35, 0.5
将计算 4 个流量 0.21 0.35 0.50 0.80
管材类型 / 管长 / 局部损失比例 / 指定管径 / 加大流量 ... (不变)
优点 - 改动最小
- 可从Excel粘贴
- 无学习成本
缺点 - 数量多时拥挤
- 不易逐个修改
- 缺少视觉反馈
B 标签芯片 (Tag Chips)

与下方坡度分母UI一致：输入回车添加、点×删除、圆角标签展示。已有成熟模式。

设计流量 Q (m³/s):
0.21 × 0.35 × 0.50 × 0.80 ×
输入回车添加 | 点 × 删除 | 支持批量粘贴
管材类型 / 管长 / 局部损失比例 / 指定管径 / 加大流量 ... (不变)
优点 - UI风格统一
- 逐个增删方便
- 视觉清晰
缺点 - 添加步骤多一步
- 大量值时较慢
- 需要新增UI代码
C 可编辑迷你表格

小型表格，每行一个Q值，带序号和删除按钮，底部有"添加行"按钮。

- 直接编辑修改
- 支持Tab键跳转
缺点 - 占用垂直空间
- Q值多时长
- 交互稍重
D 混合模式 (输入框 + 芯片)

输入框支持逗号分隔批量输入/粘贴，同时以标签芯片展示已添加的值。两种方式兼得。

设计流量 Q (m³/s):
0.21 × 0.35 × 0.50 × 0.80 ×
输入单个值回车添加 | 粘贴 "0.1, 0.2, 0.3" 批量添加 | 点 × 删除

优点 - 两种输入都支持
- Excel粘贴友好
- 视觉+效率兼得
- 逐个增删方便
缺点 - 实现稍复杂
- 输入框行为需

说明

  • 以上4种方案仅改造流量Q的输入区域，其余参数（管材、管长、局部损失、指定管径、加大流量等）保持不变
  • 单个Q值时行为与现有完全一致（向下兼容）
  • 输入多个Q后点"计算"，所有Q值共享相同的管材/管长等参数，分别计算
  • 结果展示方案将在选定输入方案后单独设计预览
