# 沿程水头损失计算说明

## 一、概述

沿程水头损失是水流在渠道中流动时，由于水流与渠壁之间的摩擦阻力而产生的能量损失。本程序采用底坡法计算沿程水头损失。

## 二、计算公式

### 2.1 基本公式

$$h_f = i \times L_{有效}$$

其中：
- $h_f$ = 沿程水头损失（m）
- $i$ = 底坡（无量纲，如 1/3000 = 0.000333）
- $L_{有效}$ = 有效长度（m）

### 2.2 有效长度计算

$$L_{有效} = \Delta S_{MC} - L_{渐变段} - \frac{L_{arc,1}}{2} - \frac{L_{arc,2}}{2}$$

其中：
- $\Delta S_{MC} = S_{MC,2} - S_{MC,1}$ = 两节点的里程MC桩号之差（m）
- $L_{渐变段}$ = 两节点之间渐变段的总长度（m），若无则为0
- $L_{arc,1}$ = 上一节点的弧长（m）
- $L_{arc,2}$ = 当前节点的弧长（m）

### 2.3 公式说明

**为什么要扣除弧长的一半？**

弧长代表弯道的长度。在弯道段，水流损失主要由弯道水头损失公式计算（见弯道损失），而非沿程损失。因此：
- 上一节点的弯道后半段不计入沿程损失
- 当前节点的弯道前半段不计入沿程损失

**为什么要扣除渐变段长度？**

渐变段有独立的水头损失计算公式，其沿程损失已在渐变段水损中计算，避免重复。

## 三、底坡数据

### 3.1 数据来源

底坡数据从表格的"底坡"列读取。表格中显示的是底坡的倒数形式（如3000），程序内部转换为真实底坡值：

$$i = \frac{1}{底坡列值}$$

例如：底坡列值 = 3000 → $i = \frac{1}{3000} \approx 0.000333$

### 3.2 底坡选取规则

计算时优先使用下游节点的底坡值：

```
slope_i = node2.slope_i if node2.slope_i > 0 else node1.slope_i
```

### 3.3 备用计算方法

当底坡数据缺失时，程序回退到曼宁公式反算水力坡降：

$$J = \left(\frac{v \cdot n}{R^{2/3}}\right)^2$$

$$h_f = J_{avg} \times L_{有效}$$

其中：
- $v$ = 流速（m/s）
- $n$ = 糙率
- $R$ = 水力半径（m）
- $J_{avg}$ = 上下游断面的平均水力坡降

## 四、计算示例

### 示例数据

| 参数 | 节点1 | 节点2 |
|-----|------|------|
| 里程MC（m） | 1000.0 | 1100.0 |
| 弧长（m） | 10.0 | 5.0 |
| 底坡 1/i | 3000 | 3000 |

渐变段长度 = 0（无渐变段）

### 计算过程

1. **里程差**
   $$\Delta S_{MC} = 1100.0 - 1000.0 = 100.0 \text{ m}$$

2. **有效长度**
   $$L_{有效} = 100.0 - 0 - \frac{10.0}{2} - \frac{5.0}{2} = 100.0 - 5.0 - 2.5 = 92.5 \text{ m}$$

3. **底坡**
   $$i = \frac{1}{3000} = 0.000333$$

4. **沿程损失**
   $$h_f = 0.000333 \times 92.5 = 0.031 \text{ m}$$

### 有渐变段的情况

若两节点之间有渐变段长度 = 20m：

$$L_{有效} = 100.0 - 20.0 - 5.0 - 2.5 = 72.5 \text{ m}$$

$$h_f = 0.000333 \times 72.5 = 0.024 \text{ m}$$

## 五、与其他损失的关系

### 5.1 水面线推求公式（顺推法）

$$Z_{下} = Z_{上} - h_f - h_j - h_w - h_{渐}$$

其中：
- $Z_{上}$ = 上游水位（m）
- $Z_{下}$ = 下游水位（m）
- $h_f$ = 沿程水头损失（m）
- $h_j$ = 局部水头损失（m）
- $h_w$ = 弯道水头损失（m）
- $h_{渐}$ = 渐变段水头损失（m）

### 5.2 总水头损失

$$h_{总} = h_f + h_j + h_w$$

注：渐变段损失单独显示在渐变段行，不计入节点的总水头损失。
