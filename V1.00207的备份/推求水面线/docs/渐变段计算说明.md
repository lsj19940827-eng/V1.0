# 渐变段计算说明

## 一、渐变段概述

渐变段是连接不同断面形式或尺寸的水工建筑物之间的过渡段。当水流从一种断面过渡到另一种断面时，需要渐变段来平滑过渡，减少水头损失。

## 二、渐变段识别规则

### 2.1 需要渐变段的情况

| 序号 | 情况描述 | 说明 |
|-----|---------|------|
| 1 | 隧洞/渡槽 → 明渠 | 总是需要渐变段，不检查底宽差异 |
| 2 | 明渠 → 隧洞/渡槽 | 总是需要渐变段，不检查底宽差异 |
| 3 | 不同结构子类型 | 如：隧洞-圆形 → 隧洞-圆拱直墙型 |
| 4 | 明渠不同子类型 | 如：明渠-梯形 → 明渠-矩形 |
| 5 | 同一明渠类型但不同流量段 | 不同设计流量的断面 |

### 2.2 不需要渐变段的情况

| 序号 | 情况描述 | 说明 |
|-----|---------|------|
| 1 | 倒虹吸相邻 | 倒虹吸水损已包含进出口损失 |
| 2 | 完全相同的结构类型 | 如：隧洞-圆形 → 隧洞-圆形（同流量段） |
| 3 | 断面特征尺寸相同 | 底宽/直径/半径换算后相等 |

### 2.3 断面特征尺寸换算规则

为统一比较不同类型断面的尺寸，采用"特征宽度"概念：

- **直径 D** → 特征宽度 = D
- **半径 R** → 特征宽度 = 2R（换算为直径）
- **底宽 B** → 特征宽度 = B

例如：渡槽U形半径1m 与 圆形隧洞直径2m 视为相同尺寸。

## 三、渐变段长度计算

### 3.1 基本公式

$$L = k \times |B_1 - B_2|$$

其中：
- $L$ = 渐变段长度（m）
- $k$ = 长度系数
  - 进口：$k = 2.5$
  - 出口：$k = 3.5$
- $B_1$ = 起始断面水面宽度（m）
- $B_2$ = 末端断面水面宽度（m）

### 3.2 约束条件

#### 渡槽

| 类型 | 约束条件 |
|-----|---------|
| 进口 | $L \geq 6 \times h_{设计}$（6倍渠道设计水深） |
| 出口 | $L \geq 8 \times h_{设计}$（8倍渠道设计水深） |

#### 隧洞

| 类型 | 约束条件 |
|-----|---------|
| 进口 | $L \geq \max(5h, 3D)$（5倍渠道水深 或 3倍洞径取大值） |
| 出口 | $L \geq \max(5h, 3D)$（5倍渠道水深 或 3倍洞径取大值） |

## 四、渐变段水头损失计算

### 4.1 总损失公式

$$h_{渐} = h_{j1} + h_f$$

其中：
- $h_{渐}$ = 渐变段总水头损失（m）
- $h_{j1}$ = 局部水头损失（m）
- $h_f$ = 沿程水头损失（m）

### 4.2 局部水头损失

$$h_{j1} = \xi_1 \frac{|v_2^2 - v_1^2|}{2g}$$

其中：
- $\xi_1$ = 局部损失系数（查表K.1.2）
- $v_1$ = 起始断面流速（m/s）
- $v_2$ = 末端断面流速（m/s）
- $g$ = 重力加速度（9.81 m/s²）

#### 局部损失系数表（表K.1.2）

| 渐变段形式 | 进口 $\xi_1$ | 出口 $\xi_1$ |
|-----------|-------------|-------------|
| 曲线形反弯扭曲面 | 0.10 | 0.20 |
| 直线形扭曲面 | 0.20 | 0.30 |
| 圆弧直墙 | 0.30 | 0.50 |
| 八字形 | 0.35 | 0.65 |
| 直角形 | 0.40 | 0.75 |

### 4.3 沿程水头损失（平均值法）

$$h_f = i \times L$$

其中：
- $i$ = 平均水力坡降
- $L$ = 渐变段长度（m）

## 五、明渠段自动插入

### 5.1 触发条件

当两个建筑物（隧洞/渡槽）之间的里程差大于两个渐变段长度之和时，自动插入明渠段。

$$\Delta S_{MC} > L_{出口渐变段} + L_{进口渐变段}$$

### 5.2 插入内容

满足条件时，自动插入3行：

1. **出口渐变段**：前一建筑物的出口渐变段
2. **明渠段**：连接两个渐变段的明渠
3. **进口渐变段**：后一建筑物的进口渐变段

### 5.3 明渠段参数确定

弹窗提供两个选项：

1. **复制上游明渠参数**（推荐）
   - 自动查找上游最近的明渠节点
   - 复制其断面参数（底宽、水深、边坡、糙率、底坡等）

2. **手动输入参数**
   - 用户填写：名称、结构形式、底宽、水深、边坡、糙率、底坡
