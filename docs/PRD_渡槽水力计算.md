> **版本**: v1.0  
> **日期**: 2026-02-22  
> **状态**: 已实现  
> **所属系统**: 渠系建筑物水力计算系统 V1.0

## 1. 背景与目标

### 1.1 背景

渡槽是渠系工程中跨越河流、道路、山谷等障碍物的输水建筑物。本模块提供渡槽槽身过水断面的水力设计计算功能，依据 **GB 50288-2018《灌溉与排水工程设计标准》** 第 9.4.1 条，完成断面尺寸确定、水力要素计算、超高验证及流速校核。

### 1.2 目标

- 支持 **U 形断面**（底部半圆 + 直立侧墙）和 **矩形断面**（可选倒角）两种槽身形式
- 自动搜索最优断面尺寸（最小槽身面积 / 深宽比约束）
- 同时计算设计流量与加大流量两种工况
- 提供详细计算过程输出、断面图预览、DXF 图纸导出、Word 计算书导出

### 1.3 用户角色

| 角色 | 描述 |
|------|------|
| 渠道设计工程师 | 输入设计参数，获取渡槽断面尺寸及水力计算成果 |

## 2. 规范依据

### 2.1 GB 50288-2018 第 9.4.1 条

水力设计内容应包括选择槽身纵向底坡、确定槽身过水断面尺寸、通过水面衔接计算确定渡槽底部纵向各部高程。

### 2.2 条文 9.4.1-1（流速推荐值）

> 槽身底坡应为缓坡（排洪渡槽除外），槽内设计流速宜为 **1.0 m/s ~ 2.5 m/s**。

- **实现方式**：流速作为推荐值，超出范围时输出提示信息（warning），但不阻止计算。

### 2.3 条文 9.4.1-2（超高控制条件）

> - **矩形断面**：设计流量时超高不应小于槽内水深的 1/12 加 0.05m；加大流量时超高不应小于 0.10m。
> - **U 形断面**：设计流量时超高不应小于槽身直径的 1/10（即 R/5）；加大流量时超高不应小于 0.10m。

- **实现方式**：超高为硬性控制条件，不满足时计算失败并给出明确错误信息和建议。

### 2.4 加大流量比例（规范表）

| 设计流量 Q (m³/s) | 加大比例 |
|-------------------|---------|
| Q < 1             | 30%     |
| 1 ≤ Q < 5         | 25%     |
| 5 ≤ Q < 20        | 20%     |
| 20 ≤ Q < 50       | 15%     |
| 50 ≤ Q < 100      | 10%     |
| Q ≥ 100           | 5%      |

用户可指定加大比例，留空时按上表自动计算。

## 4. 水力计算核心

### 4.1 基础公式 — 曼宁公式

```
Q = (1/n) × A × R_hyd^(2/3) × i^(1/2)
V = (1/n) × R_hyd^(2/3) × i^(1/2)
R_hyd = A / P
```

其中 A = 过水面积，P = 湿周，R_hyd = 水力半径。

### 4.2 U 形断面水力要素

U 形断面由下半圆（半径 R）+ 两侧直墙（高度 f）组成。

**当 h ≤ R 时（水面在半圆部分）：**

```
θ = arccos((R - h) / R)
A = R² × (θ - sinθ × cosθ)
P = 2Rθ
```

**当 h > R 时（水面在直段部分）：**

```
A = πR²/2 + 2R × (h - R)
P = πR + 2 × (h - R)
```

**槽身总面积（用于最优搜索）：** 与过水面积公式相同，h 取总高。

### 4.3 矩形断面水力要素

**无倒角：**

```
A = B × h
P = B + 2h
```

**带倒角（角度 α，底边长 c）：**

倒角高度：`c_h = c × tan(α)`

倒角斜边长：`c_hyp = c / cos(α)`

当 h ≥ c_h 时（水深超过倒角高度）：

```
A = B × h - c × c_h
P = (B + 2h) - 2(c + c_h) + 2 × c_hyp
```

当 h < c_h 时（水深在倒角区域内）：

```
A = B × h - 2c × h + c × h² / c_h
P = (B - 2c) + 2 × (h / c_h) × c_hyp
```

### 4.4 水深反算

采用**二分法**迭代求解水深 h，使曼宁公式计算流量逼近目标流量。

| 参数 | 值 |
|------|------|
| 收敛容差 | 0.0001（相对误差） |
| 最大迭代次数 | 1000 |
| U 形搜索上界 | h_max = f + R |
| 矩形搜索上界 | 20.0 m |
| 精度终止条件 | \|h_high - h_low\| < 0.000001 |

### 4.5 U 形断面最优搜索逻辑

**未指定 R 时**，自动搜索最小槽身面积对应的 R：

| 参数 | 值 |
|------|------|
| 搜索范围 | R = 0.2 m ~ 15.0 m |
| 搜索步长 | 0.01 m |
| f/R 约束 | 0.4 ~ 0.6 |
| H/(2R) 约束 | 0.7 ~ 0.9 |
| 初拟 f/R | 0.4 |
| 优化目标 | 最小化 `calculate_u_total_area(H, R)` |

**搜索流程**（逐 R 遍历）：

1. 初拟 f/R = 0.4，计算 H_初 = R + f
2. 反算加大流量水深 h_inc
3. 计算需求总高 = max(h_inc + R/5, h_inc + 0.10)
4. 若需求总高 > H_初，则调整 f/R（钳位到 0.4~0.6 范围）
5. 向上取整总高到 2 位小数
6. 验证：加大流量超高 ≥ 0.10m，设计流量超高 ≥ R/5
7. 验证：H/(2R) 在 0.7~0.9 范围内
8. 记录满足约束且面积最小的 R

**指定 R 时**：

1. 初拟 f/R = 0.4，计算 H
2. 反算加大水深，验证超高
3. 若不满足，调整 f/R（钳位 0.4~0.6）
4. 验证两种工况超高，不满足则报错并给出建议

### 4.6 矩形断面搜索逻辑

**指定槽宽 B 时**：

1. 反算设计/加大水深
2. 设计流量总高需求：H1 = h_design + h_design/12 + 0.05
3. 加大流量总高需求：H2 = h_inc + 0.10
4. 取 H = max(H1, H2)，向上取整 2 位小数

**未指定 B 时**（按深宽比搜索）：

1. 深宽比 α = 用户值 或默认 0.8
2. B 从 0.50m 到 20.0m，步长 0.01m
3. 目标总高 H_target = B × α
4. 对每个 B：反算水深 → 计算超高需求 → 向上取整
5. 取第一个满足 H ≤ H_target 的 B（最小宽度解）

### 4.7 超高计算与验证

| 工况 | U 形断面 | 矩形断面 |
|------|---------|---------|
| 设计流量 | Fb_设计 = H - h_design ≥ R/5 | Fb_设计 = H - h_design ≥ h_design/12 + 0.05 |
| 加大流量 | Fb_加大 = H - h_inc ≥ 0.10m | Fb_加大 = H - h_inc ≥ 0.10m |

不满足任一条件即判定计算失败，返回 `success = False` 和详细错误信息。

### 4.8 流速校核

| 项目 | 要求 |
|------|------|
| 推荐范围 | 1.0 ~ 2.5 m/s |
| 超出处理 | 输出 warning_message，不阻止计算 |
| 过小提示 | "流速过小，可能造成淤积" |
| 过大提示 | "流速过大，可能造成冲刷" |

## 7. 成果输出

### 7.1 DXF 断面图导出

**文件**：`渠系断面设计/aqueduct/dxf_export.py`

**比例尺选择**：1:20 / 1:50 / 1:100 / 1:200 / 1:500

**DXF 图层设置**：

| 图层名 | 颜色 | 线宽 | 内容 |
|--------|------|------|------|
| OUTLINE | 7 (白) | 0.50mm | 槽身轮廓 |
| WATER_DESIGN | 5 (蓝) | 0.25mm | 设计水位线 |
| WATER_INCREASED | 4 (青) | 0.25mm | 加大水位线（虚线） |
| DIMENSION | 2 (黄) | 0.18mm | 尺寸标注 |
| TEXT_PARAMS | 3 (绿) | 0.18mm | 参数文字块 |

**绘制内容**：

- **U 形**：下半圆弧（40段折线近似）+ 两侧直墙 + 顶部虚线 + 水面线 + 尺寸标注（B/R/f/H）+ 参数文字块
- **矩形**：矩形框（含可选倒角线段）+ 顶部虚线 + 水面线 + 尺寸标注（B/h/H）+ 参数文字块
- **参数文字块内容**：断面类型、比例尺、输入参数（Q/n/i）、断面尺寸、设计工况（h/A/V）、加大工况（比例/Q/h/V/Fb）

**依赖**：`ezdxf` 库，缺失时 `ImportError` 提示安装。

共用工具函数来自 `渠系断面设计/open_channel/dxf_export.py`：`_add_layer`、`_add_dim_h`、`_add_dim_v`、`_add_text_block`。

### 7.2 Word 计算书导出

**格式**：`.docx`，使用 `python-docx` 生成高端咨询报告风格文档。

**文档结构**：

1. **封面**：标题"渡槽水力计算书" + 副标题（断面类型 + 设计方法）+ 页眉
2. **一、基础公式**：曼宁公式、槽宽/槽高公式（LaTeX 公式渲染）
3. **二、计算过程**：纯文本计算过程转 Word 段落（自动识别标题/公式行）
4. **三、断面图**：matplotlib 图表导出为 PNG 嵌入（宽度 14cm，DPI 150）

**依赖**：`python-docx`、`latex2mathml`、`lxml`，缺失时 InfoBar 提示安装命令。

共用工具函数来自 `渠系断面设计/export_utils.py`：`create_styled_doc`、`doc_add_h1`、`doc_add_formula`、`doc_render_calc_text`、`doc_add_figure` 等。

### 7.3 计算结果显示

支持两种显示模式：

| 模式 | 触发条件 | 内容 |
|------|---------|------|
| 简要模式 | 取消勾选"输出详细计算过程" | 输入参数汇总 + 断面尺寸 + 两工况结果 + 验证结论 |
| 详细模式 | 勾选"输出详细计算过程"（默认） | 完整14步推导过程（含每步公式代入、中间值计算） |

**详细模式计算步骤**（以 U 形为例）：

1. 输入参数
2. 断面尺寸（B=2R、f/R、H=R+f、H/B）
3. 设计水深计算（曼宁公式反算）
4. 过水面积计算（分 h≤R / h>R 两种情况）
5. 湿周计算
6. 水力半径计算
7. 设计流速计算（曼宁公式）
8. 计算流量验证（误差百分比）
9. 加大流量计算
10. 加大水深计算
11. 加大过水面积 & 湿周
12. 加大水力半径 & 流速
13. 流量校核
14. 超高计算 + 验证（两工况）

## 10. 错误处理

### 10.1 输入验证

| 条件 | 处理 |
|------|------|
| Q ≤ 0 | 阻止计算，InfoBar 提示 |
| n ≤ 0 | 阻止计算，InfoBar 提示 |
| slope_inv ≤ 0 | 阻止计算，InfoBar 提示 |
| v_min ≥ v_max | 阻止计算，InfoBar 提示 |
| 倒角参数不完整 | 弹出 `fluent_question` 对话框 |
| 深宽比 ≤ 0 | 计算引擎返回错误 |
| 指定槽宽 ≤ 0 | 计算引擎返回错误 |

### 10.2 计算错误

| 条件 | 处理 |
|------|------|
| 水深反算失败 | `success=False`，提示计算失败 |
| 超高不满足规范 | `success=False`，提示具体不满足哪条规范及建议 |
| 未找到满足约束的解 | `success=False`，提示调整参数 |
| 指定 R 过小 | `success=False`，建议增大 R 或改用自动搜索 |

### 10.3 导出错误

| 条件 | 处理 |
|------|------|
| 未计算即导出 | InfoBar 警告 |
| 缺少 ezdxf | InfoBar 提示安装命令 |
| 缺少 python-docx | InfoBar 提示安装命令 |
| 文件被占用 | InfoBar 提示关闭文件 |
| 其他异常 | InfoBar 显示错误详情 |
