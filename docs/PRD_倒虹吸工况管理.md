# 倒虹吸工况管理功能 PRD

> **已归档**：本文档内容已合并到 `倒虹吸水力计算系统/docs/倒虹吸水力计算软件 - 前端需求规格说明书 (Frontend SRS).md` v4.0 第 2 章（工况管理系统）。本文件保留供历史参考。

**归档日期**: 2026-03-06

## 功能概述
为倒虹吸水力设计面板添加工况管理功能，支持用户同时管理多个倒虹吸设计方案进行对比。

## 核心需求

### 1. 工况存储
- 存储位置：`data/siphon_cases/` 目录
- 文件格式：`工况名.siphon.json`
- 支持导入导出：`.siphon.json` 和 `.json` 格式

### 2. UI 布局
- 左侧边栏：工况列表（宽度可调节，默认 200px）
- 右侧主面板：倒虹吸设计界面
- 工具栏位置：边栏顶部（新建、导入按钮）

### 3. 工况操作
- **新建**：自动命名为"工况1"、"工况2"等
- **重命名**：双击工况名或右键菜单
- **复制**：自动命名为"原名称_副本"
- **删除**：无需确认，直接删除（文件可从系统恢复）
- **导入/导出**：支持单个工况的分享
- **拖拽排序**：支持手动调整工况顺序

### 4. 交互行为
- 单击切换工况
- 双击重命名
- 右键菜单：重命名、复制、导出、删除
- Delete 键删除当前工况

### 5. 自动保存
- 切换工况时自动保存
- 参数修改后 2 秒自动保存
- 计算完成后自动保存
- 关闭面板时自动保存

### 6. 示例数据处理
- 首次打开自动创建"工况1"
- 新工况自动添加纵断面示例数据（13个节点，包含竖曲线和折点）
- 示例数据来源：`倒虹吸水力计算系统/resources/导入纵断面dxf示例.dxf`
- 示例数据特点：
  - 桩号范围：0.00m ~ 249.87m（已归零处理）
  - 包含4个竖曲线（ARC，R=5m）
  - 包含1个折点（FOLD）
  - 高程范围：48.87m ~ 113.84m
- 保存时标记 `longitudinal_is_example: true`
- 工况切换时检测无纵断面数据则自动添加示例

## UI 设计

### 工况列表样式（Fluent Design）
- 圆角卡片式列表项（border-radius: 4px）
- 渐变色选中效果（#0078D4 到 #106EBE）
- 悬停半透明背景（rgba(0, 0, 0, 0.05)）
- 图标+背景双重标识当前工况

### 右键菜单
- 使用 qfluentwidgets 的 RoundMenu
- 带流畅动画（MenuAnimationType.DROP_DOWN）
- 图标：FluentIcon.EDIT, COPY, SHARE, DELETE

## 技术实现

### 核心文件
1. `case_manager.py` - 工况管理器（CRUD 操作）
2. `case_sidebar.py` - 工况侧边栏 UI（Fluent Design）
3. `panel.py` - 集成工况管理到主面板

### 关键方法
- `_on_case_selected()` - 切换工况
- `_add_example_longitudinal()` - 添加示例数据
- `_mark_dirty()` - 标记数据修改
- `_do_autosave()` - 执行自动保存
