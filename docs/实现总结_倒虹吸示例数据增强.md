## 关键设计决策

1. **示例数据不保存**: 避免工况文件污染，每次打开空工况都重新生成示例
2. **自动转换**: 用户编辑后自动转为用户数据，无需手动操作
3. **确认对话框**: 导入DXF时总是确认，防止误操作覆盖数据
4. **视觉标识**: 灰色斜体 + 黄色提示标签，清晰区分示例数据和用户数据
5. **最小化修改**: 复用现有的 `_sync_nodes_to_segments()` 方法，保持代码简洁
