Logo 替换说明
将对话框左上角的默认 Qt 图标替换为项目 Logo

问题：当前对话框 dlg = QWidget() 未调用 setWindowIcon()，所以显示的是 Qt 默认图标。

修复：在对话框创建后加一行代码，使用项目已有的 logo.svg / logo.ico 作为窗口图标。效果如上方模拟窗口标题栏所示。

代码位置：渠系断面设计/water_profile/panel.py 第 4332 行后

# 设置窗口图标（复用主窗口 logo）
import os
from PySide6.QtGui import QIcon
_res = os.path.join(os.path.dirname(os.path.dirname(os.path.abspath(__file__))), "resources")
for _f in ("logo.ico", "logo.svg"):
  if os.path.exists(os.path.join(_res, _f)):
    dlg.setWindowIcon(QIcon(os.path.join(_res, _f)))
    break
