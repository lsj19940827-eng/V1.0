# 软著代码生成工作流

将指定项目的全部源代码去掉注释、docstring、空行后，合并输出为Word文档，精确60页（前30页+后30页，每页50行）。

## 参数

用户需提供：
1. **项目根目录**（默认: `C:\Users\大渔\Desktop\有压管道设计_独立版`）
2. **源文件列表**（按合并顺序）
3. **输出文件名**

## 步骤

1. 遍历源文件列表，逐文件读取，过滤掉：
   - 空行
   - `#` 开头的注释行
   - `"""`/`'''` 包裹的 docstring（单行和多行）

2. 合并后统计纯代码总行数

3. 提取前1500行（前30页）+ 后1500行（后30页），连续拼接

4. 生成Word文档，关键排版参数：
   - A4纸: 21cm × 29.7cm
   - 上下边距: 2.54cm，左右边距: 2.0cm
   - Normal样式: Consolas 9pt，段前段后间距 = 0
   - **固定行距: `Cm(0.466)`** — 实测精确适配50行/页
   - 前后30页之间**不加分页符**（加了会多1页）

5. 输出 `.docx` 文件

## 已验证的配置（有压管道水力计算系统）

- 脚本: `有压管道设计_独立版/tools/extract_copyright_code.py`
- 运行: `python tools/extract_copyright_code.py`
- 纯代码: 4912行
- 输出: 精确60页

## 复用到其他项目

修改脚本中的 `SOURCE_FILES` 列表即可。核心排版参数（行距0.466cm、Consolas 9pt、边距）不需要改动。
